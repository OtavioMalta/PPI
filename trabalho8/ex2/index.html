No Ex1, é utilizado a função JSON.parse para converter a resposta textual para JSON

Já no Ex2, é definido o tipo de resposta que será recebido antes, utilizando a propriedade xhr.responseType = 'json'. Portando, não é necessário realizar a conversão do objeto como no Ex1, sendo necessário apenas utilizar a propriedade xhr.response, a qual retornará um objeto JSON.

No Ex3, por sua vez, é realizado uma operação onde um objeto JSON será enviado ao servidor. Primeiramente é criado o objeto JS e definido no cabeçalho da requisição o tipo do conteúdo a ser enviado (xhr.setRequestHeader("Content-Type", "application/json")). Em seguida, é utiliazdo o JSON.parse como no Ex1 para converter o objeto recebido em JSON. E por fim, o objeto JS é convertido para JSON e a requisição é enviada.
